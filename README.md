# Intelligent CPU Scheduler Simulator

A Python-based CPU Scheduling Simulator built for CSE316 - Operating Systems (LPU).

## Features
- FCFS (First Come First Serve) Scheduling
- SJF (Shortest Job First) Scheduling
- Round Robin Scheduling
- Priority Scheduling
- Gantt Chart Visualization
- Performance Metrics: Waiting Time, Turnaround Time, Response Time, CPU Utilization, Throughput

## How to Run
1. Install dependencies:
pip install customtkinter matplotlib

2. Run the simulator:
python main.py

## Project Structure
- scheduler/ - Contains all scheduling algorithms
- gui/ - Contains the GUI application
- main.py - Entry point of the application

## Technology Used
- Python 3.x
- CustomTkinter (GUI)
- Matplotlib (Gantt Chart)
